# @fermi-ventures/launchkit-ui

> ⚠️ **Status: Planned** — This package is not yet implemented.

## Planned Features

Shared React UI components for SaaS applications, including:

- Portal shell and navigation components
- Dashboard layouts and grids
- Form components (built on shadcn/ui)
- Data tables with sorting/filtering
- Modal and dialog patterns
- Loading states and skeletons
- Toast notifications
- Empty states and error boundaries
